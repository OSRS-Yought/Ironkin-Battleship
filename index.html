Build the board

Upload the board screenshot. It gets sliced into one tile per cell, so each tile carries its own art and can be shuffled without losing it. Tile names load from tiles.seed.json when the count matches. This replaces every existing tile and clears all claims.

Shuffle

Randomizes which task sits in which cell. Claims, hits, and splashes travel with the task, not the cell. Run this after both teams have placed ships.

Reset shots

Sets every tile back to unfired and removes all claims. The tiles and their art stay. Use this if you want to run a second round on the same board.
