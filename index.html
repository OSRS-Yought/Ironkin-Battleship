Build the board

Upload the board screenshot. It gets sliced into one tile per cell, so each tile carries its own art and can be shuffled without losing it. Tile names load from tiles.seed.json when the count matches. This replaces every existing tile and clears all claims.

Sprite % is how much of each cell, measured from the top, holds the item icon. The rest is the baked-in name, which gets thrown away — the app draws names as real text instead. 60 suits a board like the one you sent. Turn it up if icons are getting clipped, down if you see leftover text.

Fetch icons from the OSRS Wiki

Replaces each tile's sliced art with the real item sprite from the wiki — transparent, high resolution, no baked-in text. Lookup names come from tiles.seed.json, matched by tile name. This does not touch claims, hits, or splashes, so it is safe to run mid-event. Takes about a minute for 100 tiles.

Shuffle

Randomizes which task sits in which cell. Claims, hits, and splashes travel with the task, not the cell. Run this after both teams have placed ships.

In-game names

Discord names show everywhere by default. Set a RuneScape name here and it replaces the Discord name on tiles, the crew list, and both exports. Discord overwrites its own name field on every login; this one it never touches. Blank the field to fall back to Discord.

Crew roster

Only these Discord IDs can load the board or change anything. Enforced in the database, so it holds even if someone skips the site and hits the API directly. In Discord: Settings → Advanced → Developer Mode, then right-click a member → Copy User ID. One ID per line.

Put a name after the ID and that person becomes assignable straight away, before they have ever opened the site — they get a Discord default avatar and show up in every "add someone" list. When they do sign in, their real name and avatar take over and every tile they were already on stays assigned. ID with no name just grants access.

Reset shots

Sets every tile back to unfired and removes all claims. The tiles and their art stay. Use this if you want to run a second round on the same board.
